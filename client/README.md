# TODO App frontend

This is the React client for the TODO APP. It is built with Vite and communicates with the backend API at `http://localhost:3001/api/todos`.

## Features

- View active tasks
- Add new todo items with title and optional description
- Edit existing tasks inline
- Toggle task completion status
- Delete tasks


## Setup

1. Open a terminal in the `client` folder.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the URL shown in the terminal (`http://localhost:5173`).

## Notes

- The frontend expects the backend to be running on port `3001`.
- The main frontend entry is `src/App.jsx`.
